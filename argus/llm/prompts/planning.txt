Output JSON matching this schema:

    {

      "type": "discover",
      "goal": "What to find",
      "agent": "scout",
      "depends_on": []
    },
    {
      "id": 2,
      "type": "extract",
      "goal": "Details to extract",
      "agent": "deep_dive",
      "depends_on": [1]
    }
  ],
  "estimated_sources": 20,
  "estimated_time_minutes": 60
}

Valid agent types: scout, deep_dive, verification, synthesis
Valid task types: discover, extract, verify, synthesize
